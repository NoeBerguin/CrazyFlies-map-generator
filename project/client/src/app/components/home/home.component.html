Third Year Final Project

What is our third year final project?

Our goal this semester was to explore an unknown area using crazyflie drones and then draw a map of the terrain they've explored so that users can see it on their screen. We also included the option to simulate the exploration using the ARGOS software. The simulation is used in order to give the user a general idea of the actual exploration.

Requirements

This project is made to run on a Linux machine and uses the ARGOS software to run the simulation. For help on how to install Ubuntu, ARGOS and other dependencies, please refer to out github. The link is under the useful links. Right here

How does the project work?

The first thing to do on the application tab is to make sure the server and the database are up and running. To do so, run the container with: docker-compose up --build

On the application tab, you are greeted with two tabs inside a drawer: Mission Options & Mission Mode. The drawer can be closed with the hamburger-button on the top left corner of the application tab (on the nav bar).

The "Mission Options" tab contains all the saved missions, the logs of the actual mission and a save button to save the current mission.

On the "Mission Mode" tab, you will find all the neccessary buttons to control your explorations. You first have to choose between the type of mission you want to launch: "Simulation" (ARGOS simulation) or "Real Drones" (the actual drone). Once you've selected the mission type, the mission state buttons will appear. If you selected the simulation mode, the ARGOS application should open and you can simply click play to begin the exploration.

Simulation Mode

On the "Simulation" Mode, by simply click on start to begin the mission. A list of drones should appear and all their related information can be show or hidden by clicking the buttons "Show" and "Hide". You can also show a particular drone's information by clicking on a specific drone section. The button "Stop" stops the drones and the button "R2B", which stands for "Return to base", will make the drones go back to base.

Real Drones Mode

The "Real Drones" mode requires you to scan for drones first. You can do that by clicking on the "Scan" button. Once you've found some drones, you can connect to them by clicking the connect button. You must connect to them individually by opening the according section. Once connected, you can "Start", "Stop", "R2B" (Return to base) and "Identify" the drones. Each button is self explanatory.

Useful tips

On the "Videos" tab, shortcuts are provided to navigate through the multiple videos. By scrolling on the different labels, you can switch through the different video sections. You can also use the keys "h" and "l".

On the "Application Tab", on the bottom right side of the map, you will find a "+" and "-" button. Use those to zoom in and out and visualize the map better!
